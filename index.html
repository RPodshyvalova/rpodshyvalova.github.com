Player 1 use keys w - for move up, s - for move down, a - move to the left, d - move to the right, z - set X or O
Player 2 use keys arrow Up - for move up, arrow Down - for move down, arrow Left - move to the left, arrow Right - move to the right, 9 - set X or O
Player with X symbol must do the first step
